# exercici-git-botiga

Aquest projecte consisteix en la creació d'una botiga online utilitzant HTML, CSS i JavaScript. L'objectiu és aprendre i posar en pràctica conceptes de desenvolupament web, incloent la maquetació de pàgines, l'estilització amb CSS i la interactivitat amb JavaScript. El projecte inclourà funcionalitats bàsiques com la visualització de productes, afegir articles a la cistella i gestionar el procés de compra.

## Guia pas a pas

1. **Crear l'estructura de carpetes i fitxers**
	- Crea una carpeta pel projecte.
	- Dins la carpeta, crea els fitxers `index.html`, `styles.css` i `script.js`.

2. **Maquetar la pàgina principal amb HTML**
	- Dissenya la pàgina amb una llista de productes.
	- Inclou seccions per la cistella i el resum de la compra.

3. **Afegir estil amb CSS**
	- Dona format visual a la botiga: colors, tipografies, marges, etc.
	- Fes que la pàgina sigui atractiva i fàcil d'utilitzar.

4. **Implementar la interactivitat amb JavaScript**
	- Permet afegir productes a la cistella.
	- Calcula el total de la compra.
	- Mostra i actualitza la cistella en temps real.

5. **Provar i millorar la botiga**
	- Revisa que totes les funcionalitats funcionin correctament.
	- Fes proves d'usuari i millora l'experiència segons sigui necessari.

---
Si tens dubtes o vols afegir noves funcionalitats, pots modificar aquest projecte i experimentar amb HTML, CSS i JavaScript!
